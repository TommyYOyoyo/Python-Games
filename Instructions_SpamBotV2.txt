How to use SpamBot V2:

1. Install Python.
2. Install pyautogui using the command "pip3 install pyautogui" in the terminal.
3. Run "spambotv2.py" in the terminal using the following command: "python3 <FILE_PATH>".
   *Don't forget to replace the <FILE_PATH> with the path of the file "spambotv2.py" on your computer.
4. Have fun!
